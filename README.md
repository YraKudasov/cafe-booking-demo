# Кафе «Тёплый угол» — демо бронирования столиков

Демо автоматизированной системы создания сайтов: свободная запись нескольких гостей на один час.

- `index.html` — сайт с формой брони (лимит: 5 столиков в час)
- `Code.gs` — бэкенд на Google Apps Script: хранит брони как события Google Календаря и проверяет вместимость перед созданием

Сайт: https://yrakudasov.github.io/cafe-booking-demo/
